# Research

Alt som handler om å forstå problemrommet, brukerne og markedet.

## Undermapper

- `competitive-analysis/` — Sammenligning av eksisterende produkter (Feedly, Matter, Readwise, Perplexity, etc.)
- `user-research/` — Innsikt om målgruppen, intervjuer, personas
- `market/` — Markedstrender, størrelse, muligheter

## Spørsmål vi prøver å svare på

- Hvem er brukerne og hva er deres nåværende arbeidsflyt for å holde seg oppdatert?
- Hva gjør eksisterende løsninger bra/dårlig?
- Hva er den udekkede plassen Linspo kan eie?
